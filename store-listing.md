# TOEIC Trail Store Listing Draft

## App Name
TOEIC Trail

## Short Description
多益 900 分衝刺學習 App，內建 5000+ 學習卡與 14000+ 練習題。

## Full Description
TOEIC Trail 是為 3-6 個月多益衝刺設計的英文學習 App。內建 5000+ 單字、片語、搭配詞與考試情境學習卡，以及 14000+ 題文法、字彙、閱讀與聽力情境練習題。

核心功能：
- 12 個多益能力單元
- Part 5 文法與詞性題
- Part 6 補文邏輯題
- Part 7 郵件、公告、備忘錄閱讀題
- 聽力情境與最佳回應題
- 5000+ 單字、片語、搭配詞學習卡
- 14000+ 可抽題題庫池
- 錯題本與弱點分析
- 粗估分數與 900 分讀書計畫
- 每日 30 題與模擬測驗模式
- XP、愛心、連勝、成就系統
- 手機 PWA，可加入主畫面
